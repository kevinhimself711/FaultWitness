# I-0008 — K3s, GPU and Isolated Runtime Foundation

## Status

In progress after passing EVAL-G01-001 and the dedicated I-0008 activation commit. Server mutation is limited to the frozen project-owned K3s/runtime foundation and must preserve the pre-bootstrap Docker baseline.

Implementation checkpoint: the pinned K3s service is active on the private server and Helm, gVisor, the NVIDIA container runtime, and the Kata runtime are installed. The repository now contains the pinned deployment manifests, infrastructure command surface, coexistence checks, runtime/network smoke definitions, and focused unit tests. This checkpoint intentionally lands the main deployment work without claiming that EVAL-G01-002 has passed.

Deferred from this checkpoint: the complete NetworkPolicy matrix, embedded-etcd restore rehearsal, project-only rollback rehearsal, clean idempotent reinstall, and candidate-bound full Eval rerun. I-0008 therefore remains `in_progress`.

## Objective

Create a reproducible Kubernetes and runtime foundation on the shared host without changing the behavior or exposure of existing Docker workloads.

## Scope

- Pinned single-node K3s and Helm, embedded-etcd snapshots, encrypted Secrets, namespaces, quotas, limits, and default-deny NetworkPolicies.
- No Traefik or ServiceLB; all services remain private and management uses a controlled tunnel.
- Real runc, gVisor, Kata, and RTX 4090 capability smoke with an exact rollback runbook.

## Non-goals

No multi-node HA, public ingress, GPU-in-sandbox combination, or G06 Sandbox security claim.

## Test and Eval

EVAL-G01-002 compares before/after Docker inventory and health, scans listeners/routes, executes the allow/deny matrix, restores etcd, and runs each RuntimeClass/GPU smoke.

## Pass criteria

- runc, gVisor, Kata, and GPU smoke: all pass or the Iteration fails.
- Existing Docker stop, recreation, port, network, and health regression: 0.
- Unexpected public listener: 0; NetworkPolicy matrix: 100% expected.
- Clean bootstrap and project-only rollback are reproducible from the candidate.
